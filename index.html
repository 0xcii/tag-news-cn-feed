    ERC-6551用例之区块链游戏
    ERC-6551 是一个为每个 ERC-721 代币（NFT）提供智能合约账户的系统。相关的智能合约被称为代币绑定账户（Token Bound Account），缩写为 TBA，表明它与 ERC-721「绑定」。代币绑定账户的控制权属于所链接 NFT 的持有者。这意味着 ERC-6551 可被视为一种建立特定 NFT 持有者，专有的合约账户的技术。它可被认为是一种能够将 NFT 变成钱包的技术——本质上就是「NFT as wallet」。由于它向后兼容 ERC-721 NFT，因此可以应用于所有遵守 ERC-721 标准的现有 NFT。  ( 3 min )
